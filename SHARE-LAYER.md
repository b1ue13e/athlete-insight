# 报告分享层实现

## 三种分享格式

### 1. 教练版摘要

**用途**: 发给教练，一屏内快速了解

**格式**:
```
刘宇航 · 主攻 · 校联赛第3轮

结论：上限在，失误拖后腿（72分）
主问题：发球失误连续3场高于均值
对比：低于均值（-5）
可信度：中（快速模式）

训练处方 P1：
发球：保守落点发球训练
量：15分钟·3组×20球/组
达标：连续10次发球不失误，且落点在目标区域
```

**操作**: 点击"复制文本"即可粘贴到微信/邮件

---

### 2. 图片卡片

**用途**: 社交媒体分享，展示训练成果

**样式**:
```
┌─────────────────────────┐
│   ATHLETE INSIGHT       │
│                         │
│          78             │
│      综合评分           │
│                         │
│   上限在，失误拖后腿    │
│                         │
│   刘宇航 · 主攻         │
│   校联赛第3轮           │
│                         │
│   ─────────────────     │
│   核心问题              │
│   发球失误偏多          │
│                         │
│   训练动作              │
│   保守落点发球训练      │
│                         │
│   2024/3/23             │
└─────────────────────────┘
```

**操作**: 
- 点击"下载图片"保存 PNG
- 点击"复制文案"获取配文

---

### 3. 完整报告 (TXT)

**用途**: 存档，打印，详细复盘

**包含内容**:
- 运动员和比赛信息
- 综合评估
- 四维得分
- 关键洞察
- 优势/需改进
- 训练处方（全部）
- 数据可信度说明

**操作**: 点击"下载报告"保存 TXT 文件

---

## 使用方式

在报告页 `/analysis/[id]/v2` 右下角有一个悬浮的分享按钮：

```
┌───────────────┐
│               │
│               │
│      [分享]   │  ← 点击打开分享面板
│               │
└───────────────┘
```

点击后弹出分享面板，三种格式一目了然。
